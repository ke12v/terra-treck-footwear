# Deploy to GitHub Pages (no local Node required)

1) Upload the contents of this folder to your GitHub repo (not the ZIP itself).
2) In GitHub: Settings → Pages → Source: GitHub Actions
3) Wait for Actions to finish (green check).
4) Your site URL:
   https://ke12v.github.io/terra-treck-footwear/

Notes:
- Vite `base` is already set to `/terra-treck-footwear/`.
- Build output is `dist/public` (this repo is configured that way).
